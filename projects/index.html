Word Arithemtic API

This is an API endpoint that gets an operation type and two variables. The operation type specifies the arithemtic operation to perform on the operands. The complexity in this project lies in the fact that the operation can be a farly long sentence which describes the operation to perform, and in some cases, refer to the operands by their order or label (e.g. remove the first number from the variable y), or even contain the variables required for the arithmetic operation.
