QR Code Generator App

Project QR-Go, built as my final project during Zuri Training, is a web application that can be used to generate specific QR codes; in this case, for upload and share ticket to an event easily, or to share the address to a private event easily and securely. The codes can be controlled and monitored from the user account. It also has the feature to scan all types of QR codes.

This is an API endpoint that gets an operation type and two variables. The operation type specifies the arithemtic operation to perform on the operands. The complexity in this project lies in the fact that the operation can be a fairly long sentence which describes the operation to perform, and in some cases, refer to the operands by their order or label, or even contain the variables required for the arithmetic operation.
